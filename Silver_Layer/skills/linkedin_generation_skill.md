# Skill: LinkedIn Generation

## Purpose
Generate professional LinkedIn posts from business content in the vault.

## Inputs
- Source content: Path to approved content file
- Post type: Announcement/Insight/Question/Story
- Target audience: Professional/Technical/General
- Tone: Professional/Casual/Inspirational

## Outputs
- LinkedIn post text (max 3000 characters)
- Hashtags (3-5 relevant tags)
- Call-to-action (optional)
- Engagement hooks (questions, polls)

## Process

1. **Analyze Source Content**
   - Read the approved content file
   - Extract key messages and value propositions
   - Identify main themes and takeaways

2. **Determine Post Structure**
   - **Announcement**: News/launch format with clear CTA
   - **Insight**: Thought leadership with actionable advice
   - **Question**: Engagement-focused with open-ended question
   - **Story**: Narrative format with personal/business journey

3. **Generate Post Content**
   - Hook (first 2 lines to grab attention)
   - Body (main content with line breaks for readability)
   - Value proposition (what reader gains)
   - Call-to-action (next step for reader)

4. **Add Engagement Elements**
   - Relevant hashtags (research trending tags)
   - Question to encourage comments
   - Tag relevant people/companies (if applicable)

5. **Format for LinkedIn**
   - Use line breaks for readability
   - Emojis sparingly (1-3 max)
   - Keep paragraphs short (2-3 lines)
   - Bold key points if needed

## Success Criteria
- Post is under 3000 characters
- Hook is compelling and relevant
- Content provides clear value
- CTA is specific and actionable
- Hashtags are relevant and not excessive

## Error Handling
- **Source content too long**: Summarize key points
- **Unclear message**: Request clarification or use generic template
- **Missing context**: Add disclaimer or request more info

## LinkedIn Best Practices
- Post during business hours (9 AM - 5 PM local time)
- Use 3-5 hashtags maximum
- Include a question to drive engagement
- Keep it authentic and personal
- Add value, don't just promote
